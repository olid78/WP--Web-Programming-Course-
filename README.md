# WP – Web Programming Course

This repository contains all assignments, projects, and practice work completed as part of the **Web Programming (WP) course**.

The purpose of this repository is to keep all course-related work in one organized place so that any assignment or project can be easily reviewed when required.

## Repository Structure

Each folder in this repository represents a specific topic, assignment, or project covered in the course.  
Folders are numbered to reflect the order in which the topics were introduced.

Example:
- `01-portfolio-website-html-css` – Portfolio website built using HTML and CSS
- `02-simple-js-task` – A simple JS Quiz App
